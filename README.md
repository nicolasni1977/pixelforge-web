# PixelForge

A browser-based Photoshop-style image editor. Single self-contained `index.html` — no build step, no dependencies, runs offline once loaded.

## Live demo

Deployed on Vercel — URL will appear here after the first deploy.

## What's inside

`index.html` is the editor itself (~130 KB, all CSS + JS inlined). The other files are artifacts from the multi-agent generation pipeline that produced this app:

| File | Purpose |
|---|---|
| `index.html` | The editor — open in any modern browser |
| `design_spec.json` | Design system, layout grid, components — input to the generator |
| `task_assignment.json` | Decomposed feature list from the orchestrator |
| `verifier_report.json` | Final acceptance scorecard from the verifier agent |
| `*.review.html` | Human-readable renderings of the above |

## Features

- **11 tools**: Move (V), Brush (B), Pencil (N), Eraser (E), Spray (A), Fill (G), Text (T), Marquee (M), Lasso (L), Magic Wand (W), Crop (C)
- **Layer system**: add / duplicate / delete / merge / reorder with drag-and-drop, per-layer opacity + visibility
- **Selection-aware painting**: marquee or lasso constrains brush / spray / eraser / fill to the selected region (Photoshop-style)
- **Selection ops**: cut / copy / paste / delete (Ctrl+X / Ctrl+C / Ctrl+V / Delete) within the selection
- **History panel** with undo / redo (Ctrl+Z / Ctrl+Shift+Z)
- **Filters**: brightness, contrast, blur, sharpen, grayscale, invert, sepia
- **Export** to PNG / JPEG, save / load as `.pixelforge` JSON
- **Keyboard shortcuts** throughout, mobile toolbar fallback under 768 px

## How it was built

Generated by the [Auto Webapp Agent Team](https://github.com/anthropics/claude-code) v1 pipeline: Orchestrator → Planner → Generator → Verifier, iterating until the verifier scored ≥ 80 / 100. Hybrid model setup: local Qwen 3.6 (35B) for generation, Claude Sonnet for verification.

## Run locally

```bash
# Just open index.html in a browser — no server needed.
xdg-open index.html        # Linux
open index.html            # macOS
start index.html           # Windows
```
